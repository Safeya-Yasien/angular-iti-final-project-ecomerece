# 🛒 Angular E-Commerce Project

A high-performance, modular e-commerce solution featuring a **Customer Storefront** and a **Robust Admin Dashboard**. This project is designed for a team of six, utilizing a feature-based architecture to ensure parallel development and scalability.

---

## 🚀 Project Features

### **Storefront (Client Side)**

- **Dynamic Browsing:** Responsive homepage with category-based filtering and search.
- **Product Discovery:** Detailed product pages with image galleries and stock status.
- **Cart System:** Persistent shopping cart with real-time quantity updates.
- **Checkout Flow:** Secure multi-step checkout optimized for Cash on Delivery (COD).
- **User Accounts:** JWT-based authentication with personalized order history.

### **Admin Dashboard (Management Side)**

- **Inventory Control:** Complete CRUD operations for categories and products.
- **Order Tracking:** Live management of order lifecycles (Pending, Shipped, Delivered).
- **Analytics:** Visual overview of store performance and sales metrics.
- **Role Security:** Protected routes and administrative guards.

## 🛠 Feature Breakdown per Member

### **1. Admin Dashboard & Categories (Admin Architect)**

- **Foundation:** Sidebar, Top Navigation, and Role-based Route Guards.
- **Category Management:** Full CRUD (Add/Edit/Delete) for product categories.
- **Deliverables:** Admin layout shell and Category API integration.

### **2. Product Management (Inventory Manager)**

- **Product CRUD:** Complex forms for product attributes (Price, Images, Attributes).
- **Inventory Tools:** Search and filtering within the admin list; stock status management.
- **Deliverables:** Product management module and image upload handling.

### **3. User Store Frontend (Storefront Lead)**

- **Experience:** Homepage, product listing pages, and responsive UI components.
- **Discovery:** Product details view and shop-wide search/filtering.
- **Deliverables:** Reusable UI library and end-to-end customer browsing flow.

### **4. Auth + Cart + Checkout (Flow Engineer)**

- **Security:** User Registration, Login, and JWT state management.
- **Cart & Checkout:** "Add to Cart" logic and the Cash-on-Delivery checkout form.
- **Deliverables:** Auth system and functional shopping cart module.

### **5. Orders & Order Management (Operations Lead)**

- **Lifecycle:** Order placement logic and user-facing order history.
- **Admin Control:** Status updates (Pending → Delivered) and tracking UI.
- **Deliverables:** Complete order workflow for both customers and admins.

### **6. Backend/API & Integration (Systems Lead)**

- **Architecture:** Database schema design and RESTful API development.
- **DevOps:** Environment setup, deployment, and code review coordination.
- **Deliverables:** API layer, DB collections, and deployment pipeline.
